# Paper Thin Maps (Visual Utility)

This utility mod redefines how maps are displayed in Item Frames, making them ultra-thin and perfectly aligned with the block surface. It’s the ultimate tool for builders who want seamless "wallpaper" maps or clean glass displays.

## Key Features:

  - **Ultra-Thin Rendering:** Reduces the map thickness to a paper-like scale, removing the bulky 3D box effect.
  - **Zero-Gap Alignment:** Adjusts the map position to sit flush against the block (including Glass), eliminating the annoying "floating" space.
  - **Auto-Invisible Frames:** Automatically hides the wooden Item Frame texture when a map is placed inside.
  - **Marker Cleanup:** Completely disables the green "Frame Markers" on maps for a cleaner, marker-free world view.

> This is a client-side and server-side compatible modification. It requires **Architectury API** to function.

## Compatibility:

  - **Loaders:** Supports both **Forge** and **Fabric**.
  - **Environments:** Perfect for creative building, map art galleries, and modern interior design.
